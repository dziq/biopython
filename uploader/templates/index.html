About

This simple application allows you to find the nucleotide sequences in FASTA files aquired from NCBI Let us assume that you are looking for SNPs in the gene sequence of bovine beta casein (10338 bp). First you have to download desired sequence as FASTA file (on NCBI site -> Send -> Choose file destination -> Format FASTA) and upload it on this site. Then enter search SNP sequence (eg. AAAGCGGGTTAA). Script will search SNP sequence through 10338 nucleotides and tell where could you find it. If there is no SNP in FASTA file script will return: Not found. Moreover it will serch for reverse complement sequence to desired SNP
